Aplikasi absensi menggunakan qrcode <br>
dibuat menggunakan :
<ol>
    <li>laravel</li>
    <ul>
        <li>laravel breeze starter template</li>
        <li>eloquent orm</li>
        <li>laravel spatie permissions</li>
        <li>laravel splade</li>
    </ul>
    <li>vue js</li>
    <ul>
        <li>vue3 qr reader</li>
    </ul>
    <li>axios</li>
    <li>DBMS mysql</li>
    <li>tailwind css</li>
</ol><br>
